JDBC
----------------------
-JDBC stands for java data base connectivity.
-It is a specification for developing database apps with java programming language.

Database application
----------------------
-An application that communicates with database is known as database application.

Application
----------------------
-An app is a program in which we interact with on the desktop.

Database
----------------------
-A database is a software and it is an organised collection of data.
-data organised in db in form of tables here table contains fields and records.
-Application communicating with database by using SQL.
-SQL stands for structured query language.it is used by an application to communicate with database.
-Application is a frontend where as database is a backend.


                          SQL request
Application(Front end)---------------------->Database(backend)
   |                                          |
   |__________________________________________|
                 <-------------- SQL response

Database s/w contains 2 parts:
1)Database application(Ex:SQL prompt)
2)Database (Ex:Oracle)

Client
----------------------------
-A client is a software that sends the request to a server to get the response.

Server
---------------------------
-A server is s/w that recieves request from the client,
process the request ,constructs the response and sends the response back to client.
